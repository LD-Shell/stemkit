Data Processing & Statistics

Popular

Plot Digitizer

Extract data points from a graph image by clicking or tracing curves, then export the numbers as CSV.

Data Cleaner

Clean, normalize, and log-transform messy CSV datasets, then download the tidied file.

Stat Calculator

Run t-tests, ANOVA, correlation, and non-parametric tests with effect sizes, CIs, and assumption checks.

Error Bar Gen

Compute mean, SD, SEM, median, IQR, and confidence intervals per group, then plot and export error bars.

Outlier Detector

Flag anomalies in your data using Z-score and interquartile-range (IQR) methods.

Curve Fitter

Fit linear, polynomial, exponential, power, or log models and read off coefficients, R², and RMSE.

Plot Builder

Overlay multiple datasets into one figure, style it fully, and export PNG/SVG or matplotlib code.

XVG Visualizer

Plot GROMACS .xvg and CSV logs interactively; select columns, smooth, and export the figure or Python code.

Computational Chemistry

3D Inspector

View molecular structures and trajectories (PDB, GRO, XYZ) in an interactive 3D WebGL viewer.

Coordinate Manipulator

Translate, center, and measure bounding boxes for PDB and GRO coordinate files.

Energy Conversions

Convert energy, length, and other units — Hartree, eV, kcal/mol, kJ/mol, cm⁻¹ — with CODATA constants.

HPC Script Gen

Generate ready-to-run SLURM job scripts and GROMACS command templates for HPC clusters.

Writing & Citations

Equation Formatter

Build LaTeX equations and matrices visually with a live KaTeX preview, then copy the code.

Visual LaTeX Tables

Paste a table from Excel, edit and merge cells visually, and copy clean LaTeX (booktabs) output.

DOI to BibTeX

Paste a DOI and get a ready-to-use BibTeX citation resolved from Crossref.

BibTeX Deduplicator

Scan a .bib library and merge duplicate entries by DOI and title.

BibTeX Sanitizer

Fix messy BibTeX exports: standardize capitalization, repair braces, and clean fields.

Journal Abbrev

Convert full journal names to their ISO 4 / CASSI standard abbreviations.
